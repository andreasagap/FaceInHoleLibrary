# FaceInHoleLibrary


## Setup

### Gradle

Add this to your project level `build.gradle`:


```groovy
allprojects {
  repositories {
    jcenter()
    maven { url 'https://jitpack.io' }
  }
}
```


Add this to your app `build.gradle` :

```groovy
dependencies {
  compile 'com.github.andreasagap:FaceInHoleLibrary:v1.0'
}
```
